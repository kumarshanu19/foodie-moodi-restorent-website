Foodie Moodie - Restaurant . Cafe . Bar

Created: 14 2019

By: Kumar Shanu

Email: er.kumarshanu@gmail.com

Foodie Moodie is an ultimate setup for your food category websites. It is designed in a very flexible way so that you can customize it in any of your food business like restaurants, cafes, bars, chefs, stores etc. etc.

[Layout CSS Customization]

visual layout and totally easy properties defined in elements folder

1. Body

2. Header / .navbar / 7 color styles

3. Slider / #myCarausel, #myCarausel2 / .Sliding1, .testimonials

4. Content / .section / .section1 - .section5 for diffrent patterns

5. Content / .container / .container to .container3 for diffrent width

6. Welcome Content / .section .section1

7. Our Services / .section .section3

8. Reservation Box / .box-reservation

9. Menu / .section .section4

10. Tabs / .menubox - .specialbox

11. Subscribe Newsletter / .subscribe-cover

12. Team / .section .team

13. Grill Fire / .section5

14. Footer / .footer

15. Color scheme comparisons for any element

- default = light scheme

- inverse = dark scheme

- primary = red scheme

- warning = orange scheme

- info = blue scheme

- success = green sheme

- danger = brown scheme
