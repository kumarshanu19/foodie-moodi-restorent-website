Animation Effects

82 Transition Effects Which will make your site look Cool!
Just add - animate/animated - to your object class and next to it any effect you want from the list.

-animate = scroll load to trigger - when object would be in view.
-animated = load to trigger at once - when page would be load.
-infinite = put it after animate/animated in object class. you'll have the effect in loop.
-Speed and Timing = right now it's in 2 seconds setup. You can customize it easily.
